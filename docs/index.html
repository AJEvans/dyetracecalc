Introduction

This code builds a website which calculates dye tracing dispersion coefficients. It demonstrates the C# Model-View-Controller (MVC) framework. It builds a website with a form which is populated with parameters from a database. The form can be adjusted by the user and then submitted to calculate a dispersion coefficient.

Building and testing the source code

To run the site:

  • Download the the repo as a zip and unzip it on your machine.
  • Download and install Visual Studio (VS) Code.
  • Install the latest .Net Software Development Kit (SDK) (this code uses .Net 5; C# 9).
  • Open VS Code and install the C# for VS Code extension from Microsoft. You may also find the MSBuild tools useful.
  • In VS Code, go File -> Open Workspace... and open the file dyetracecalc.code-workspace, which should be in the unzipped directory (directory probably called dyetracecalc-main).
  • Agree to any boxes that appear asking you to do anything in the way of adding assets or restoring code.
  • Open a terminal in VS Code (Terminal -> New Terminal).
  • At the terminal change directory (cd command) to the DyeTraceCalcMvc directory, e.g. cd c:\downloads\dyetracecalc-main\DyeTraceCalcMvc.
  • Enter dotnet run.
  • Open a web browser with the URL http://localhost:5000

Troubleshooting

If the site doesn’t run, you may need to rebuild the code to download libraries. At the Terminal prompt, type:

dotnet build

Then try running the code again.

If your browser complains of security issues it may be automatically redirecting to the secure https version; you may need to install a developers’ certificate: at the terminal, enter

dotnet dev--certs https --trust

Close and reopen your web browser.

 

Before using this code to run actual dye tracing experiments, please note the caveats in the comments for the file /DyeTraceCalcMvc/Calc/Calculator.cs.

The full repo for the source is on GitHub if you need it.
