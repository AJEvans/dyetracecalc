Introduction

This code builds a website which calculates dye tracing information. It demonstrates the C# Model-View-Controller (MVC) framework.

Building and testing the source code

To run the site:

  • Download the repo and unzip it on your machine.
  • Install the latest .Net libraries (this code uses .Net 5; C# 9).
  • Download and install Visual Studio (VS) Code.
  • Install the C# for VS Code extension from Microsoft. You may also find the MSBuild tools useful.
  • Open up the Workspace file dyetracecalc.code-workspace
  • Open a terminal in VS Code.
  • At the terminal change directory (cd command) to the DyeTraceCalcMvc directory.
  • Enter dotnet run.
  • Open a web browser with the URL http://localhost:5000

If the browser complains of security issues it may be automatically redirecting to the secure https version; you may need to install a developers’ certificate: at the terminal, enter

dotnet dev--certs https --trust

Close and reopen your web browser.

 

Before using this code to run actual dye tracing experiments, please note the caveats in the comments for the file /DyeTraceCalcMvc/Calc/Calculator.cs.
